# Residential Energy Consumption Analysis

## Project Overview

This project was completed as part of the IST 687 – Introduction to Data Science course in the M.S. in Applied Data Science program at Syracuse University.

The project analyzes residential energy consumption data to identify usage patterns and the factors that influence electricity demand. The analysis was performed using R to explore trends, build predictive models, and visualize energy usage.

---

## Objective

- Analyze residential energy consumption data
- Explore factors affecting electricity usage
- Build predictive models using R
- Create clear visualizations to communicate findings

---

## Tools & Technologies

- R
- ggplot2
- dplyr
- Linear Regression
- Statistical Analysis
- RStudio

---

## Project Workflow

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature exploration
- Predictive modeling
- Model evaluation
- Data visualization

---

## Key Results

- Identified important factors affecting residential energy consumption.
- Built predictive models to estimate electricity usage.
- Created visualizations to explain energy consumption patterns.
- Demonstrated practical applications of statistical analysis using R.
