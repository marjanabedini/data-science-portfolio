# Data Science Salary Insights

## Project Overview

This project was completed as part of the IST 719 – Information Visualization course in the M.S. in Applied Data Science program at Syracuse University.

This individual project explores global data science salary data to identify salary patterns across different job roles, experience levels, company sizes, and geographic locations using data visualization techniques.

---

## Objective

- Analyze global data science salary data
- Explore salary differences across multiple factors
- Create clear and informative visualizations
- Communicate insights through visual storytelling

---

## Tools & Technologies

- R
- ggplot2
- Data Visualization
- Data Analysis
- Statistical Graphics

---

## Project Workflow

- Data cleaning and preparation
- Exploratory data analysis
- Visualization design
- Pattern identification
- Insight communication

---

## Key Results

- Identified salary trends across experience levels and job titles.
- Compared salaries across countries and company sizes.
- Visualized compensation differences using multiple chart types.
- Presented findings through effective visual storytelling.
